<h1>Exchange Bot: Privacy Policy</h1>

<h2>1. Data Collection and Processing</h2>

<p>Exchange Bot operates primarily as a stateless utility. To fulfill user requests, we temporarily process command arguments 
(such as stock tickers, fiat currency codes, and cryptocurrency symbols) provided during interactions with the bot.
  We do not permanently store or log private user text, user profiles, or server message histories.</p>

<h2>2. Third-Party Data Transmission</h2>

<p>To provide real-time market insights, the financial tickers you input are transmitted to the following third-party application 
programming interfaces (APIs):

<ul>
  <li>Yahoo Finance API (Query1): For equity and stock market data.</li>
  
  <li>Kraken API: For cryptocurrency pricing and market data.</li>
  
  <li>Frankfurter API (frankfurter.dev): For foreign fiat currency exchange rates.</li>
</ul>

No Personally Identifiable Information (PII), Discord usernames, or unique Discord IDs are transmitted to these third parties.</p>

<h2>3. Data Security</h2>

<p>Because Exchange Bot does not retain user databases or personal history, your data cannot be exposed or sold. 
Any operational data processed in memory during a command is discarded immediately upon displaying the results to the Discord channel.</p>

<h2>4. Changes to This Policy</h2>

<p>If Exchange Bot introduces features requiring a permanent database (such as personalized asset tracking or watchlists), this
policy will be updated, and explicit user consent will be requested within the application prior to storing user data.</p>
