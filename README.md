<h1>Exchange Bot</h1>
<img src="https://i.postimg.cc/YqXSHtnb/Exchange-Bot-Profile-Pic.png" style="width: 25%; height: 25%;">

A bot that allows for quick checking for stock and crypto prices in Discord, along with exchange rates.

OAuth2 Permissions Required:
<ul>
<li>bot</li>
<li>application.commands</li>
</ul>

> [!CAUTION]
> Do not use this bot for any financial decisions or trading. The entire thing is reliant on a free API that may lag behind, give inaccurate information, or go offline.
